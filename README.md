# 🎬 K-Means Movie Recommendation System

A Machine Learning web application that recommends movies based on genres using K-Means Clustering.

Built using:
- Streamlit
- Scikit-learn
- K-Means Clustering
- TMDB Movie Dataset

---

# Live Demo

https://k-meansmovie-recommendation.streamlit.app/

---

# Project Overview

This project groups movies into clusters using K-Means Clustering and recommends movies based on selected genres.

The application:
- Detects and handles outliers
- Performs feature scaling
- Applies K-Means clustering
- Uses business logic for selecting K value
- Recommends top movies by genre

---

# Features

- 🎥 Genre-based movie recommendation
- 📊 K-Means clustering
- 📈 Outlier detection and handling
- ⚡ Feature scaling using StandardScaler
- 🎯 Business logic-based clustering
- 🌐 Interactive Streamlit UI
